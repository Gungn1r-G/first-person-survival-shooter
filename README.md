# First-Person Survival Shooter

## Overview
A Unity first-person survival shooter prototype developed using C# and Unity. The project focuses on gameplay programming, player interaction, combat mechanics, UI systems, and survival gameplay.

## Features
- First-person player movement
- Shooting mechanics
- Enemy interactions
- UI systems
- Gameplay scripting
- Object-oriented programming
- Debugging and iterative testing

## Technologies
- Unity
- C#
- GitHub
